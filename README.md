# 📊 Mobile Sales Dashboard (Power BI Project)

## 📌 Project Overview

This project showcases an interactive **Mobile Sales Dashboard** built using Microsoft Power BI. The objective was to transform raw and messy sales data into meaningful insights through data cleaning, transformation, and visualization.

---

## 🧹 Data Cleaning & Preparation

The dataset contained inconsistencies and errors. The following steps were performed:

* Removed duplicate records
* Handled missing values (Customer, Price, Quantity)
* Converted data types (Date, Numeric fields)
* Standardized date formats
* Removed invalid entries (negative quantity, incorrect dates)
* Recalculated Total (Price × Quantity)

---

## 📐 Data Modeling

* Structured dataset for efficient reporting
* Created relationships (if applicable)
* Optimized columns for filtering and analysis

---

## 🧮 DAX Measures

Key metrics created using DAX:

* **Total Sales**
* **Total Quantity**
* **Transactions Count**
* **Average Price**
* **MTD (Month-to-Date Sales)**
* **SPLY (Same Period Last Year)**
